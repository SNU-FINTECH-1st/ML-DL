# ML-DL
이 모델 사투리 마이 묵었다 아이가


###  모델 돌리기전 필수사항!
 > * 꼭 google cloud sdk를 먼저 설치해야함.
   * google cloud tokken은 유료임을 주의!
   * 무료로 하고 싶으면 다른 tts 사용하시면 됩니다.
   * finetuning 한 KOBART 는 용량 떄문에 모델 업로드하지 못했음.
   * using_KOBART로 finetuning하는것을 권함.
